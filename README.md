# Rutas Verdes

![](releases/example.png)

## Introduccion

#### Pagina web de las rutas de las busetas de sincelejo<br><br>Sus caracteristicas son:

- Acceso a las rutas de las busetas 
- Busqueda de puntos de interes
- Ruta sugerida para tomar el recorrido
- **Para que no tomes mas mototaxis**
- **Para que ahorres plata**
- **Para que pierdas el tiempo esperando y viajando : ]**<br><br>
